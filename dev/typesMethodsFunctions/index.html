LTVsystems.LTIreceiversO — Type
z = LTIreceiversO([R₁,R₂,…Rₙ],𝐩ᵣ)

Create an LTI Omnidirectional Receiver by calling LTIreceiversO() with the receiver position, 𝐩ᵣ and all the reflections, Rᵢ where i=1,2,…n.

Examples

using ISA, LTVsystems
using Plots
𝐩ₛ =  [0.0, 0.0]
𝐩ᵣ =  [1.0, 0.0]
p(t) = δ(t-1.0e-15,1.0e-10)
q = LTIsourcesO(𝐩ₛ, p)
α₁ = 0.7; 𝛏₁ = [1.8,0.0]
R₁ = LTIsourcesO(𝛏₁, t->α₁*q(𝛏₁,t))
z = LTIreceiversO([R₁],𝐩ᵣ)

LTVsystems.LTIsourcesO — Type
q = LTIsourcesO(𝐩ₛ, p)
R = LTIsourcesO(𝛏, r)

Create an LTI Omnidirectional Source by calling LTIsourcesO() with the source position, 𝐩ₛ and the transmisson signal, p.

Examples

using ISA, LTVsystems
using Plots
𝐩ₛ =  [0.0, 0.0]
𝐩ᵣ =  [0.0, 0.0]
p(t) = δ(t-1.0e-15,1.0e-10)
q = LTIsourcesO(𝐩ₛ, p)

Another type of sources, called as reflected sources can also be defined by calling LTIsourcesO() with reflectors position, 𝛏 and the reflected signal, given by r = α q(𝛏,t).

Examples

using ISA, LTVsystems
using Plots
𝐩ₛ =  [0.0, 0.0]
𝐩ᵣ =  [0.0, 0.0]
p(t) = δ(t-1.0e-15,1.0e-10)
q = LTIsourcesO(𝐩ₛ, p)
α₁ = 0.7; 𝛏₁ = [1.8,0.0]
R₁ = LTIsourcesO(𝛏₁, t->α₁*q(𝛏₁,t))
