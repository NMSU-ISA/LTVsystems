LTVsystems.LTIreceiverO — Type
z = LTIreceiverO([r],𝐩ᵣ)
z = LTIreceiverO(r,𝐩ᵣ)

Create an LTI Omnidirectional Receiver by calling LTIreceiverO() with the receiver position, 𝐩ᵣ and a vector of single reflection, r, provided by calling pointReflector().

Examples

using LTVsystems
𝐩ₛ =  [0.0, 0.0]
𝐩ᵣ =  𝐩ₛ
tₚ = 1.0e-06
p(t) = δn(t-tₚ,1.0e-07)
q = LTIsourceO(𝐩ₛ, p)
α = -0.7; 𝛏 = [3.75e-06c,0.0]
r = pointReflector(𝛏,α,q)
z = LTIreceiverO([r],𝐩ᵣ)

In order to observe the multiple reflections, we create an LTI Omnidirectional Receiver by calling LTIreceiverO() with the receiver position, 𝐩ᵣ and observed multiple reflection, r, provided by calling pointReflector().

Examples

using LTVsystems
𝐩ₛ =  [0.0, 0.0]
𝐩ᵣ =  𝐩ₛ
tₚ = 1.0e-06
p(t) = δn(t-tₚ,1.0e-07)
q = LTIsourceO(𝐩ₛ, p)
α₀ = -0.7; 𝛏₀ = [3.75e-06c,0.0]
α₁ = -0.7; 𝛏₁ = [1.5e-06c,0.0]
α₂ = -0.7; 𝛏₂ = [2.5e-06c,0.0]
r = pointReflector([𝛏₀,𝛏₁,𝛏₂],[α₀,α₁,α₂],[q])
z = LTIreceiverO(r,𝐩ᵣ)

LTVsystems.STATsourceD — Type
q = STATsourceD(𝐩ₛ, p)

Create a Stationary Directional Source by calling STATsourceD() with the source position, 𝐩ₛ ,a transmitted signal, $\mathsf{p}(t)$, time-varying (rotating) beam center, $\bm{b}_\mathrm{s}(t)$ and the source antenna's gain, $\mathrm{G}_\mathrm{s}(\Theta).$

Examples

using LTVsystems
𝐩ₛ =  [0.0, 0.0]
tₚ = 1.0e-06
p(t) = δn(t-tₚ,1.0e-07)
𝐛ₛ(t) = [cos(2π*10*t),0.0]/(norm(cos(2π*10*t)))
G(θ) = 𝒩ᵤ(θ, μ=0.0, σ=π/8)
q = STATsourceD(𝐩ₛ, p, 𝐛ₛ, G)
