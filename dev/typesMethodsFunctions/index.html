LTVsystems.LTIreceiverO — Type
z = LTIreceiverO([R₁,R₂,…Rₙ],𝐩ᵣ)

Create an LTI Omnidirectional Receiver by calling LTIreceiverO() with the receiver position, 𝐩ᵣ and all the reflections, Rᵢ where i=1,2,…n.

Examples

using LTVsystems
𝐩ₛ =  [0.0, 0.0]
𝐩ᵣ =  𝐩ₛ
p(t) = δn(t,1.0e-10)
q = LTIsourceO(𝐩ₛ, p)
α₀ = 0.7; 𝛏₀ = [1.8,0.0]
r = pointReflector(𝛏₀,α₀,q)
z = LTIreceiverO([r],𝐩ᵣ)

LTVsystems.LTIsourceO — Type
q = LTIsourceO(𝐩ₛ, p)
R = LTIsourceO(𝛏, r)

Create an LTI Omnidirectional Source by calling LTIsourceO() with the source position, 𝐩ₛ and the transmisson signal, p.

Examples

using LTVsystems
𝐩ₛ =  [0.0, 0.0]
p(t) = δ(t,1.0e-10)
q = LTIsourceO(𝐩ₛ, p)

Another type of sources, called as reflected sources can also be defined by calling LTIsourceO() with reflectors position, 𝛏 and the reflected signal, given by r = α q(𝛏,t).

Examples

using LTVsystems
𝐩ₛ =  [0.0, 0.0]
p(t) = δ(t,1.0e-10)
q = LTIsourceO(𝐩ₛ, p)
α₁ = 0.7; 𝛏₁ = [1.8,0.0]
R₁ = LTIsourceO(𝛏₁, t->α₁*q(𝛏₁,t))
